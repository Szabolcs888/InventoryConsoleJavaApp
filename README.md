# Inventory_ConsoleJavaApp

It is an inventory management console application that records inventory receipts, deletions, new and existing customers, and purchases. For now, it is still organized into lists, but the integration of the database and the web appearance are also in the development process.

Ez egy készletnyilvántartó konzolalkalmazás, amely regisztrálja az árukészlet bevételezését,
törlését, az új és meglévõ ügyfeleket és a vásárlásokat. Egyelõre még listákba szervezve, de fejlesztési folyamatban van az adatbázis integrálása és a webes megjelenés is.
