# Inventory_ConsoleJavaApp

This is an inventory management console application created as part of a hands-on project. __When it was made, I didn't know the clean code principles yet! I will try to rewrite it in the next few days (October-November) and make it bilingual.__ In terms of its functions, the application registers receipt and deletion of inventory, new and existing customers and purchases. For now, it is still organized into lists, but there are also plans to integrate the database and develop a simple frontend interface.
