# Inventory console Java app

This is an inventory management console application created as part of a hands-on project. In terms of its functions, it registers inventory receipts, cancellations, new and existing customers, and purchases. For now, it is still organized into lists, but there are also plans to integrate the database and develop a simple frontend interface. __In the next few days (beginning of November), I will try to make the application bilingual.__
